# Misc

The Miscellaneous settings provide additional customization options for various personalization features of your Flipper Zero. These settings can be found by pressing `UP` on the Desktop and navigating to `MNTM > Misc`.

## [Screen](Screen.md)

Configure display settings for your Flipper's screen:

- [`Display Mode`](Screen.md#display-mode): Adjust screen display settings
    - [`Dark Mode`](Screen.md#dark-mode): Toggle dark mode `ON` or `OFF`
    - [`Left Handed`](Screen.md#left-handed): Enable left-handed mode
- [`RGB Settings`](Screen.md#rgb-settings): Configure color options for RGB-capable screens (if applicable)
    - [`RGB Backlight`](Screen.md#rgb-backlight): Whether RGB backlight is installed
    - [`LCD LED 1`](Screen.md#lcd-led-1): Configure the first LCD LED color
    - [`LCD LED 2`](Screen.md#lcd-led-2): Configure the second LCD LED color
    - [`LCD LED 3`](Screen.md#lcd-led-3): Configure the third LCD LED color
    - [`Rainbow LCD`](Screen.md#rainbow-lcd): Enable rainbow LCD mode
    - [`Rainbow Speed`](Screen.md#rainbow-speed): Set the speed of the rainbow effect
    - [`Rainbow Interval`](Screen.md#rainbow-interval): Set the interval of the rainbow effect
    - [`Rainbow Saturation`](Screen.md#rainbow-saturation): Set the saturation of the rainbow effect

## [Dolphin](Dolphin.md)

Manage the stats and behavior of your digital pet:

- [`Dolphin Level`](Dolphin.md#dolphin-level): Set the level of your Dolphin
- [`Dolphin XP`](Dolphin.md#dolphin-xp): Set the XP of your Dolphin
- [`Dolphin Angry`](Dolphin.md#dolphin-angry): Set the anger/butthurt level of your Dolphin
- [`Butthurt Timer`](Dolphin.md#butthurt-timer): How quickly your Dolphin gets angry from lack of attention

## [Spoofing](Spoofing.md)

Change identification and appearance settings:

- [`Flipper Name`](Spoofing.md#flipper-name): Customize your device's displayed name (leave empty for real name), also affects BLE name and MAC, USB name and serial number
- [`Shell Color`](Spoofing.md#shell-color): Change the displayed color of your Flipper

## [VGM Options](VGM.md)

Configure settings for the Video Game Module (requires installing RGB Firmware on VGM):

- [`Install RGB Firmware`](VGM.md#install-rgb-firmware): Instructions to install the RGB Firmware to VGM
- [`Foreground`](VGM.md#foreground-background): Set the foreground color for the display (Default: Black)
- [`Background`](VGM.md#foreground-background): Set the background color for the display (Default: Orange)

## [Momentum Intro](Misc.md#momentum-intro)

Play the Momentum Firmware introduction screen that appears on initial setup.
